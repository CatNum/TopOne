## 新版本创建指引

每次开始新产品版本的设计工作时：

1. 复制上一版本目录：`cp -r docs/design/vX.X docs/design/vY.Y`
2. 更新本索引表中的版本状态
3. 在新版本目录的 `README.md` 中只记录**相对上一版本的变更内容**
