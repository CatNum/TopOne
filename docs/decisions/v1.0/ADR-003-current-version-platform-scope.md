# ADR-003 Current Version Platform Scope

## 状态

已接受

## 适用版本

- `v1.0`

## 背景

在高保真原型与当前产品范围进一步收敛后，当前版本需要把页面设计、交互确认与实现范围聚焦到同一组平台，避免原型范围与工程目标不一致。

## 决策

`TopOne v1.0` 当前版本范围聚焦 `iOS`、`iPadOS`。

具体规则：
- 当前版本高保真原型只覆盖 `iOS`、`iPadOS`。
- `iPadOS` 基于 `iOS` 页面放大适配，不单独设计独立信息架构。
- `macOS` 不纳入当前版本原型与实现范围。

## 影响

正向影响：
- 统一原型范围、需求范围与当前实现目标。
- 降低当前版本设计与实现分叉的风险。
- 让当前迭代更聚焦于移动端主流程闭环。

代价：
- `macOS` 体验延后到后续版本再单独评估。
